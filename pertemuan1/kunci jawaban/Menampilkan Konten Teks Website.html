HyperText Markup Language atau sering disebut HTML adalah bahasa standar yang digunakan untuk membuat sebuah halaman web. Bahasa HTML diakses untuk menampilkan berbagai informasi di dalam sebuah penjelajah web Internet (Browser). HTML digunakan sebagai link link antara file-file dalam situs atau dalam komputer dengan menggunakan localhost, atau link yang menghubungkan antar situs dalam dunia internet.
